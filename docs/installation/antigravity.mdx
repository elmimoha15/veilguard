# Install on Antigravity

---

## Step 1 — Add the MCP config

Create `.gemini/mcp.json` in your project folder:

```json
{
  "mcpServers": {
    "veilguard": {
      "command": "npx",
      "args": ["-y", "--package=veilguard-cli", "veilguard-mcp"],
      "env": {
        "VEILGUARD_KEY": ""
      }
    }
  }
}
```

**Alternative:** Use the Antigravity **MCP Settings Panel** → Add Server:

| Field | Value |
|-------|-------|
| **Name** | `veilguard` |
| **Command** | `npx -y --package=veilguard-cli veilguard-mcp` |
| **Environment** | `VEILGUARD_KEY=` (empty for free) |

---

## Step 2 — Add the rules file (recommended)

Download [`.antigravityrules`](https://raw.githubusercontent.com/elmimoha15/veilguard/main/templates/antigravityrules.txt) and save it to your project root.

This file tells Antigravity's AI when to call each Veilguard tool. Without it, the tools still work but won't fire automatically.

---

## Step 3 — Restart Antigravity

Close and reopen Antigravity completely.

---

## Verify

Ask the AI: *"What Veilguard tools do you have access to?"*

It should list 14 security tools.

---

## Pro key

To activate Pro, add your license key to `.gemini/mcp.json`:

```json
"env": {
  "VEILGUARD_KEY": "vg_live_xxxxxxxxxxxxxxxxxxxx"
}
```

Restart Antigravity after saving. Get a key at [veilguard.dev/pro](https://veilguard.dev/pro).

---

## Notes

- `.gemini/mcp.json` is project-level — add it to each project you want protected
- `.antigravityrules` is also project-level
- Node.js 18 or later is required
