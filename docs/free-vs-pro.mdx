## What "depth-limited" means on free

All 14 scanners run on the free tier and alert you to every vulnerability they find. The difference is the solution (and some scan depth):

| Scanner | Free | Pro |
|---------|------|-----|
| Vulnerability alerts | Every finding | Every finding |
| Fix / solution | 🔒 Locked (alert only) | Shown |
| Breach context | 🔒 Locked | Shown (which real breach each pattern caused) |
| Dependency scanning | Critical CVEs only | All severities (critical, high, medium, low) |
| Git history scan | Current files only | Full history scan |
| Supply chain check | Top 20 dependencies | All dependencies |

## What only Pro unlocks

### The fixes (and breach context)
On free, every finding shows the alert — what's wrong, where, and the real-world consequence — but the **fix/solution** and **breach context** are locked, and Veilguard prompts you to upgrade. Pro reveals the exact fix for every finding.

### The full security audit
The combined `full_audit` (security score, letter grade A+ to F, and the AI-ready one-paste fix prompt) is **Pro-only**. On free, calling it returns an upgrade prompt — no grade, no report. Free users still run the 14 individual scanners for unlimited alerts.

### Full depth on dependency and git scans
Pro shows all CVE severities (not just critical), checks all dependencies (not just the first 20), and scans your full git history for committed secrets.

## How to upgrade

1. Go to [veilguard.dev/pro](https://veilguard.dev/pro)
2. Complete checkout (Polar)
3. Your license key arrives by email
4. Open your IDE's MCP config (see [Configuration](/docs/configuration))
5. Add the key to `VEILGUARD_KEY`:

```json
{
  "mcpServers": {
    "veilguard": {
      "command": "npx",
      "args": ["-y", "--package=veilguard", "veilguard-mcp"],
      "env": {
        "VEILGUARD_KEY": "vg_live_xxxxxxxxxxxxxxxxxxxx"
      }
    }
  }
}
```

6. Restart your IDE — Pro features activate immediately

## Full audit access

The full security audit is **Pro-only and unlimited**. **Free does not get any full audit** — calling `full_audit` on free returns an upgrade prompt.

The 14 individual scanners run with no limit on any tier — free sees the alerts, Pro sees the fixes.
