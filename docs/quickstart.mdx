# Quick Start

Get Veilguard protecting your project in under 30 seconds.

---

## Prerequisite

Node.js **18 or later** must be installed. Check with:

```bash
node --version
```

---

## One command

Open your terminal, `cd` into your project root, and run:

```bash
npx veilguard-cli init
```

That's it. The init command:

1. **Installs rules files** for every supported IDE (Cursor, Windsurf, VS Code, Claude Code, Antigravity)
2. **Writes the MCP config** for every supported IDE — each gets its own config file
3. **Updates `~/.windsurf/mcp.json`** if Windsurf is installed (global config)
4. **Updates your `.gitignore`** — adds `.veilguard/` so scan caches don't get committed

Then **restart your IDE** and Veilguard is active.

---

```

  ✓ Security rules installed
    .cursorrules          (Cursor + VS Code)

  ✓ MCP config added for:
    .cursor/mcp.json      (Cursor)
    .vscode/mcp.json      (VS Code)
    .claude/mcp.json      (Claude Code)
    .gemini/mcp.json      (Antigravity)
    ~/.windsurf/mcp.json  (Windsurf — updated)

  ✅ Veilguard is ready. Restart your IDE to activate.
  Free: all 13 scanners active.
  Pro: add VEILGUARD_KEY to your MCP config → veilguard.dev/pro
```

One command sets up **every IDE at once**. The rules files and MCP configs don't conflict — each IDE only reads its own.

---

## After setup

**Restart your IDE.** The MCP server starts the next time your IDE launches.

From that point, your AI agent will:
- Scan files automatically as you build
- Stay silent if scans are clean
- Alert you in plain English if something risky is found

---

## Manual setup

If `npx veilguard-cli init` doesn't detect your IDE, or you prefer to configure manually, see the [Installation guides](/docs/installation/windsurf) for per-IDE instructions.

---

## Add your Pro key (optional)

If you have a Pro license, open the MCP config file your IDE is using and add your key:

```json
{
  "mcpServers": {
    "veilguard": {
      "command": "npx",
      "args": ["-y", "--package=veilguard-cli", "veilguard-mcp"],
      "env": {
        "VEILGUARD_KEY": "your_key_here"
      }
    }
  }
}
```

Restart your IDE after saving. Pro features activate immediately.

Get a key at [veilguard.dev/pro](https://veilguard.dev/pro).
